2017

An Improvement of Scalar Multiplication by Skew Frobenius Map with Multi-Scalar Multiplication for KSS Curve

Khandaker Md. Al-Amin Yasuyuki Nogami
IEICE Transactions December 2017

Scalar multiplication over higher degree rational point groups is often regarded as the bottleneck for faster pairing based cryptography. This paper has presented a skew Frobenius mapping technique in the sub-field isomorphic sextic twisted curve of Kachisa-Schaefer-Scott (KSS) pairing friendly curve of embedding degree 18 in the context of Ate based pairing.

Efficient Implementation of Pairing-Based Cryptography on Embedded Devices Using FPGA

Khandaker Md. Al-Amin Satoshi Tanaka Yasuyuki Nogami
International Conference on Information Security and Cryptology November 2016

This paper presents an efficient implementation of pairing-based cryptography on embedded devices using FPGA. We propose a hardware architecture for computing bilinear pairings that is suitable for resource-constrained environments.
